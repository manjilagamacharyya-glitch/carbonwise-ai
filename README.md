# ⚡ How to Run

## Local Preview

Clone the repository:

```bash
git clone <repository-url>

cd carbonwise-ai
```

Run a local server:

```bash
python -m http.server 8000
```

Open:

```text
http://localhost:8000
```

---

## Firebase Deployment

The project is configured for Firebase Hosting.

Deploy with:

```bash
firebase deploy
```

Live Demo:

https://carbonwise-ai-81a30.web.app/

---

# 🔧 Gemini AI Integration

To enable live Gemini AI:

Open:

```text
js/ai-engine.js
```

Replace:

```javascript
const GEMINI_API_KEY = "YOUR_KEY";
```

with your actual API key.

If no key is available:

✅ CarbonWise AI automatically enables **Demo AI Mode**.
